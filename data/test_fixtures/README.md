# Fixtures de Prueba

Esta carpeta contiene datos de prueba y fixtures para testing.

## Objetivo
Almacenar datos de ejemplo y casos de prueba para validar el sistema sin exponer datos reales.

## Contenido típico

- PDFs de ejemplo (anonimizados)
- Datos de prueba estructurados
- Casos edge case para testing
- Mocks y fixtures para pruebas unitarias

## Notas
- Estos archivos pueden ser compartidos (una vez anonimizados)
- Útiles para debugging y desarrollo
- Pueden incluirse en el repositorio si están anonimizados
